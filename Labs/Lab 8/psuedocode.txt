driverexam.py

create an array of correct answers
get the answers from the student
put the students' answers in a file
read a line from the file
if the line read is the same as the correct answer
	print that it's correct
	add it to a counter
if the line isn't the same as the correct answer
	print that it's incorrect
	add it to a different counter
	add it to a separate list of 'incorrect answers'
perform calculations for the average, number of right or wrong answers
print out the list of incorrect answers to the user
